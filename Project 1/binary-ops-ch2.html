Binary Operators

Binary Operators take two inputs and give an output. It is a way to test elements of a set, or operands. It is a way to filter the information that you want to have at the end of the operation or to tell the computer what you want it to do. You can add and subtract, multiply and divide. Think of binary operators as a shorthand for commands of action to take.

Examples

Operator Definition
"AND" The "AND" operator is used to link two operands together and see if they both come out to the same value.
"OR" The "OR" operator is used when you want either or both of the operands to be true or present in the data.
"NOT" The "NOT" operator is when we want to make sure that an operand is NOT in the data.
"XOR" The Exclusive OR is used when you want one of the values to be true and the other to be false.
